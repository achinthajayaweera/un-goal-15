Home Screen

Technical Description

The HTML and CSS of this page employ several key techniques for both structure and styling. The HTML uses semantic elements like "section" and `nav` to organize content logically, while external resources like Google Fonts and Font Awesome are included for custom fonts and icons. The navigation bar includes dropdowns for enhanced user interaction, and the hero section features a video background using the `video` tag, overlaid with a text box that is centered with flexbox. CSS resets the default margin/padding, and a global font is applied. Flexbox and CSS Grid are used to create responsive layouts, such as the statistics and initiative sections, ensuring adaptability across screen sizes. Interactive hover effects are applied to buttons and images for a dynamic user experience. The page uses media queries to adjust styles for smaller screens, maintaining readability and functionality on mobile devices. Finally, the footer is organized using a grid system for structured links and social icons, making the page highly functional and user-friendly.

Accessibility

Accessibility is addressed in this page through several techniques. Semantic HTML tags like 'nav', `section`, and `footer` improve screen reader navigation. The use of `alt` attributes on images provides descriptive text for users with visual impairments. Responsive design via media queries ensures the page is accessible on various devices, including mobile. The hover effects are subtle enough not to hinder users with motion sensitivity, and contrast in text and background colors ensures readability for users with low vision. Additionally, buttons and links are clear and navigable with keyboard inputs..

Technical Description

Challenges and Lessons Learned

Throughout the implementation of these web pages, key challenges included managing responsive design, content organization, and accessibility across all sections. For responsive design, CSS media queries were utilized to adjust layouts, such as grid-based data tables and navigation bars, ensuring they functioned properly on various devices. Organizing complex sections like "Health & Well-Being Impact Data" and "Mental Well-Being" required a structured approach, solved by using Flexbox and Grid to keep layouts balanced and readable. Accessibility was addressed by incorporating semantic HTML elements (e.g., `section`, `nav`) and providing `alt` attributes for images, ensuring screen reader compatibility and keyboard navigation. Styling challenges, particularly for interactive elements like hover effects on buttons and dynamic elements like dropdowns, were resolved by leveraging CSS transitions and hover states. These efforts highlighted the importance of planning for responsiveness and accessibility early in the design process, while CSS Grid and Flexbox proved essential for maintaining a flexible yet organized structure across diverse content sections.

Compliance

The pages comply with key Janet regulations by ensuring secure HTTPS connections for external resources and including privacy policy links.
